# Netflix Content Analysis Dashboard – Excel

## Project Overview

This project analyzes over 4,000 Netflix titles using Microsoft Excel to uncover trends in content genres, ratings, release years, and content-producing countries. The dashboard provides interactive visualizations and insights to help understand Netflix's content library.

## Objectives

* Analyze genre popularity and content distribution.
* Explore rating trends across Netflix titles.
* Identify release year patterns.
* Discover top content-producing countries.
* Present insights through an interactive dashboard.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning
* Data Visualization

## Key Features

* Interactive Excel dashboard.
* Genre popularity analysis.
* Rating distribution visualization.
* Release trend analysis.
* Country-wise content analysis.

## Insights Generated

* Identified the most popular content genres.
* Analyzed content ratings distribution.
* Explored trends in content releases over time.
* Highlighted leading content-producing countries.
